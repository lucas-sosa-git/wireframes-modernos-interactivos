La variable logistica es un digito que identifica la cantidad de unidades que contiene cada Bulto Continente o unidad comercial. Puede asignársele valores del 1 al 8 para unidades de consumo de medida fija y el 9 para unidades de consumo de medida variables.
Ejemplo con la variable Logistica 1 para la caja que contiene 6 unidades de un GTIN (EAN) o unidad comercial. Ejemplo con la variable Logistica 2 para la caja que contiene 12 unidades del mismo GTIN (EAN) o unidad comercial. En este caso serían 2 presentaciones o bultos con la misma unidad comercial o GTIN contenido. Se podrían generar hasta 8 presentaciones de un mismo GTIN. La variable Logistica 9 se utiliza únicamente para unidades de consumo de medida variable, ejemplo frutas, carnes, pan, etc. ( todo producto que se venda al peso).
